Data-Set Describtions

Packages Used

Analyzing air pollution levels and trends using Python and data visualization techniques.

  • Pandas: For data manipulation and processing pollutant data.
  • NumPy: For numerical operations and handling pollutant values.
  • Matplotlib: For generating static visualizations like bar and pie charts to display pollutant distribution.
  • Seaborn: For advanced visualizations and enhancing chart aesthetics related to pollutant analysis.
  • Warnings: To suppress non-critical warnings during the analysis.

Summary Information About the Data:

The dataset provides air pollution data for the United States, tracking the Air Quality Index (AQI) for several key pollutants: NO2 (Nitrogen Dioxide), O3 (Ozone), SO2 (Sulfur Dioxide), and CO (Carbon Monoxide). The data is collected from various states and cities across the U.S.

Key columns in the dataset include:

  • State: The U.S. state where the data was recorded.
  • City: The specific city within the state where measurements were taken.
  • NO2 AQI, O3 AQI, SO2 AQI, CO AQI: The AQI values representing pollution levels for each pollutant.
  • Date: The date on which the data was collected.

This dataset is useful for analyzing air pollution trends over time, comparing pollutant levels across different regions, and understanding the effects of various pollutants on air quality in the U.S.
